# recipe-explorer-47437-47446

This repository contains a Slidev-based Recipe Explorer. The app lists recipes and lets you view details (ingredients and steps) with a modern "Ocean Professional" theme.

- App path: `slidev_frontend/`
- To run:
  - cd slidev_frontend
  - npm install
  - npm run dev
  - Visit http://localhost:3000

To add more recipes, see `slidev_frontend/README.md`.
